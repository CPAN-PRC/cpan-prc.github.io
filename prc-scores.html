PRC Dist Scoring for unassigned distributions

Generated 2015-04-17 07:50:01 UTC

This is a sortable & searchable table of the CPAN distributions that are currently candidates (ie not yet assigned) for the CPAN pull request challenge. There are more dists with a github repo given in the metadata, but this only contains dists where the repo actually exists.

The RDeps column is the number of reverse dependencies that the distribution is — this is the number of other distributions on CPAN that depend on the dist in question. The Score is calculated according to a scoring scheme described in a blog post.
